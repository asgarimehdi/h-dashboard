# Plan: Health House (خانه بهداشت) Boundary Seeder

## Summary
Create a PostGIS Voronoi-based boundary seeder for type 9 (خانه بهداشت) units, generating polygons from their lat/lng points clipped to county boundaries. Follows the same approach as the deleted `AbharCenterBoundarySeeder.php`.

## File to Create
- `database/seeders/HealthHouseBoundarySeeder.php`

## Approach
For each county (region with type='county'):
1. Query all type 9 units with lat/lng in that county
2. Build a PostGIS geometry collection of points
3. Generate Voronoi polygons using `ST_VoronoiPolygons(ST_Collect(...))`
4. Dump individual polygons with `ST_Dump(...)`
5. Match each polygon back to its unit point via `ST_Contains`
6. Clip each polygon to the county boundary via `ST_Intersection`
7. Insert each result into `boundaries` table
8. Update each unit's `boundary_id`

## Implementation

### SQL Pattern (per county)
```sql
WITH points AS (
    SELECT
        unnest(ARRAY[<unit_ids>]) AS unit_id,
        unnest(ARRAY[<points>]) AS geom
),
voronoi AS (
    SELECT (ST_Dump(
        ST_VoronoiPolygons(ST_Collect(ARRAY[<points>]))
    )).geom AS polygon
)
SELECT
    p.unit_id,
    ST_AsText(
        ST_Multi(
            ST_Intersection(
                ST_Transform(v.polygon, 4326),
                (SELECT boundary FROM boundaries WHERE id = <county_boundary_id>)
            )
        )
    ) AS wkt
FROM voronoi v
JOIN points p ON ST_Contains(v.polygon, p.geom)
WHERE ST_Intersects(
    ST_Transform(v.polygon, 4326),
    (SELECT boundary FROM boundaries WHERE id = <county_boundary_id>)
)
```

### PHP Seeder Structure
```php
<?php
namespace Database\Seeders;

use App\Models\Unit;
use App\Models\Region;
use Illuminate\Database\Seeder;
use Illuminate\Support\Facades\DB;

class HealthHouseBoundarySeeder extends Seeder
{
    public function run(): void
    {
        $counties = Region::where('type', 'county')
            ->whereNotNull('boundary_id')
            ->get();

        foreach ($counties as $county) {
            $units = Unit::where('region_id', $county->id)
                ->where('unit_type_id', 9)
                ->whereNotNull('lat')
                ->whereNotNull('lng')
                ->pluck('id', 'id')
                ->toArray();

            if (count($units) < 3) continue; // Need at least 3 points for Voronoi

            // Build arrays for unnest
            $unitIds = array_keys($units);
            $points = Unit::whereIn('id', $unitIds)
                ->get()
                ->map(fn($u) => "ST_SetSRID(ST_MakePoint({$u->lng}, {$u->lat}), 4326)")
                ->toArray();

            // Run Voronoi query
            $sql = $this->buildVoronoiQuery($unitIds, $points, $county->boundary_id);
            $results = DB::select($sql);

            // Insert boundaries and link to units
            foreach ($results as $row) {
                $boundaryId = DB::table('boundaries')->insertGetId([
                    'boundary' => DB::raw("ST_GeomFromText('{$row->wkt}', 4326)"),
                    'created_at' => now(),
                    'updated_at' => now(),
                ]);

                Unit::where('id', $row->unit_id)->update(['boundary_id' => $boundaryId]);
            }
        }
    }

    private function buildVoronoiQuery(array $unitIds, array $points, int $boundaryId): string
    {
        $ids = implode(',', $unitIds);
        $pts = implode(',', $points);

        return <<<SQL
            WITH points AS (
                SELECT
                    unnest(ARRAY[{$ids}]) AS unit_id,
                    unnest(ARRAY[{$pts}]) AS geom
            ),
            voronoi AS (
                SELECT (ST_Dump(
                    ST_VoronoiPolygons(ST_Collect(ARRAY[{$pts}]))
                )).geom AS polygon
            )
            SELECT
                p.unit_id,
                ST_AsText(
                    ST_Multi(
                        ST_Intersection(
                            ST_Transform(v.polygon, 4326),
                            (SELECT boundary FROM boundaries WHERE id = {$boundaryId})
                        )
                    )
                ) AS wkt
            FROM voronoi v
            JOIN points p ON ST_Contains(v.polygon, p.geom)
            WHERE ST_Intersects(
                ST_Transform(v.polygon, 4326),
                (SELECT boundary FROM boundaries WHERE id = {$boundaryId})
            )
        SQL;
    }
}
```

### Register in DatabaseSeeder
Add `$this->call(HealthHouseBoundarySeeder::class);` after the existing seeder calls.

2. Check that boundaries are created: `SELECT COUNT(*) FROM boundaries;`
3. Check that type 9 units have boundary_id: `SELECT COUNT(*) FROM units WHERE unit_type_id = 9 AND boundary_id IS NOT NULL;`
