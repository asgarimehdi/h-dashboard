# Plan: Redesign Unit Map Filters — Horizontal Compact Layout

## Summary
Replace the current toggle-heavy filter panel with a compact horizontal filter bar. Use single-select dropdowns for county, type, and parent center. Search stays as text input.

## File to Modify
- `resources/views/livewire/maps/unit.blade.php`

## Changes

### 1. PHP Class — Simplify properties

**Replace array properties with single values:**
```php
// Before:
public array $selectedRegions = [];
public array $selectedTypes = [5, 6, 7, 9];

// After:
public ?int $selectedRegion = null;
public ?int $selectedType = null;
public ?int $selectedParent = null;
```

**Update `fetchUnits()` logic:**
- If `$selectedRegion` is null → return empty (no county = no units)
- If `$selectedRegion` is set → filter by `region_id`
- If `$selectedType` is set → filter by `unit_type_id`
- If `$selectedParent` is set → filter health houses by `parent_id`

**Update `updated*()` hooks:**
- `updatedSelectedRegion()`, `updatedSelectedType()`, `updatedSelectedParent()`, `updatedSearch()`

### 2. Blade View — Horizontal filter bar

Replace the entire `.unit-menu` div with a horizontal bar:

```blade
<div class="filter-bar bg-base-100/60 rounded-box">
    {{-- Search --}}
    <input type="text" wire:model.live.debounce.300ms="search"
           placeholder="جستجو..." class="input input-bordered input-sm flex-1 min-w-0" />

    {{-- County dropdown --}}
    <select wire:model.live="selectedRegion" class="select select-bordered select-sm">
        <option value="">شهرستان</option>
        @foreach ($regions as $region)
            <option value="{{ $region['id'] }}">{{ $region['name'] }}</option>
        @endforeach
    </select>

    {{-- Type dropdown --}}
    <select wire:model.live="selectedType" class="select select-bordered select-sm">
        <option value="">نوع</option>
        @foreach ($types as $type)
            <option value="{{ $type['id'] }}">{{ $type['name'] }}</option>
        @endforeach
    </select>

    {{-- Parent center dropdown --}}
    <select wire:model.live="selectedParent" class="select select-bordered select-sm">
        <option value="">مرکز والد</option>
        @foreach ($parentCenters as $center)
            <option value="{{ $center['id'] }}">{{ $center['name'] }}</option>
        @endforeach
    </select>
</div>
```
